## 📌 Overview

This repository serves as a **professional technical reference database** for architects, engineers, contractors, and procurement specialists.

It contains **watermarked PDF drawings** showing profile cross-sections for high-quality aluminum doors and windows manufactured in **Foshan, China**.

> 🔒 **Important:** All PDFs are watermarked for intellectual property protection.
> 
> 📅 **Regular Updates:** This library is actively maintained. New technical drawings and series are added regularly. Star this repository to stay notified.

## 🔒 Why Watermarked PDFs?

We provide **watermarked PDFs for reference only** to:
- Protect our manufacturer's intellectual property
- Prevent unauthorized commercial use
